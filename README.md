# SaimCPP - Competitive C++ Platform

A competitive C++ learning platform for students with real-time code compilation, leaderboards, and progress tracking.

## Features
- 100+ C++ problems (Arrays 1D, 2D, 3D)
- Real-time code compilation via Piston API
- Google OAuth authentication
- Department & batch-based leaderboards
- Progress tracking and streaks
- Supabase backend for data persistence

## Tech Stack
- Frontend: Vanilla JS, HTML5, CSS3
- Editor: Monaco Editor
- Backend: Supabase (PostgreSQL)
- Code Execution: Piston API
- Deployment: Vercel
